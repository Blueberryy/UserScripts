# GreasyFork: ナビゲーションバーにカテゴリを追加する

---

## 機能の説明

強化するGreasyForkナビゲーションバー, ユーザーリストを追加、ユーザーのお気に入り、ユーザーコンソールなど....


---

## その他の指示

さらにアイテムが追加されました,混雑したナビゲーションバーが表示されます,スクリプトを使用して修正できます [GreasyFork Header Style Fix](https://greasyfork.org/scripts/473269)

使用を強くお勧めします **[GreasyFork 糊裱匠](https://greasyfork.org/zh-CN/scripts/497346)** ナビゲーションバーを修復および美化することができます

---

## ありがたい
