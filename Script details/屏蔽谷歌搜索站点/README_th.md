## บล็อกบางเว็บไซต์จาก Google Search

- **ช่วยให้ผู้ใช้กรองผลการค้นหาที่ไม่ต้องการออก**
- **ไซต์บล็อกอินพุตแบบกำหนดเอง**
- **รูปแบบ** `"-zhihu" "-baidu"` ใช้เพื่อบล็อกเนื้อหาการค้นหาของ Google
